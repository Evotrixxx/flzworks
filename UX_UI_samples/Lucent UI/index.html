LUCENT UI

Welcome to **Lucent UI**, an interactive showcase of a liquid-glass design system.

Every component in this library is built with semantic HTML and styled with premium CSS glassmorphism. A **WebGL refraction shader** runs in the background.

As you move your cursor, the WebGL lens follows and **automatically morphs** its size, aspect ratio, and corner roundness to match the exact boundaries of the card you are hovering over, creating a realistic physical depth effect.
