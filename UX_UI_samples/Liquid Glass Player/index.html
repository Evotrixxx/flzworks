Liquid Glass UI System

A premium glassmorphic component library driven by real-time WebGL fluid simulation and SVG displacement filters.

Refractive Depth

Achieved via a fixed WebGL canvas rendering custom noise-based distortion behind elements, creating organic lens effects under the cursor.

Physical Imperfection

Utilizes SVG turbulence and displacement mapping bound to CSS filters, generating microscopic refraction on the glass sheets.

Real-time Control

Dynamically binds CSS inputs to WebGL fragment shader uniforms, offering complete control over fluid viscosity, aberration, and lighting.
