What changes — and why
The parallax stays. Every other structural decision is reconsidered.
Element Status Current Proposed Reason
Parallax background Keep 4-layer mouse-tracked depth scene, #05060b base Identical — not touched The scene is the hero. Everything else frames it.
Navigation Refine Floating pill, backdrop-blur, border-radius 999px, centered Full-width top bar, transparent over parallax, scrolls to semi-opaque. Active link = 1px underline only, no bg highlight. Pill nav competes with the parallax scene. A flat bar disappears into the composition.
Hero text layout Restructure Text bottom-pinned, FLZ left / info block right, asymmetric FLZ WORKS vertically centered across full height. Status row top, descriptor + counts middle, CTAs bottom. Spreads content across the full viewport. Bottom-pinning collapses all tension. Vertical spread makes the parallax work harder — eye travels through the whole scene.
FLZ / WORKS treatment Keep FLZ outline stroke, WORKS solid white — same size Same — preserved exactly. Centering makes it more monumental. The outline/solid split is the strongest visual element. No change needed.
Scroll progress bar Move Bottom edge of the floating nav pill Top of viewport, 1px full-width line More readable position, doesn't compete with nav shape.
Archive card grid Redesign hud-card: backdrop-filter blur, gradient bg, individual gap spacing Contact-sheet grid: 1px gap lines (no spacing), image fills the card as background at 30% opacity, lifts on hover. Numbered index overlay. The blur on hud-card fights the parallax blur. Contact-sheet grid reads as a deliberate design choice — archival, not technical UI.
backdrop-filter on cards Remove blur(24px) saturate(160%) on every archive card None — flat dark surface Multiple blur layers compound GPU cost and reduce parallax clarity.
Signals / Instagram Restructure Dense clip-squircle grid, 5 columns Horizontal filmstrip, scroll-overflow, square items without clip-path Filmstrip reads as a feed, implies chronology. Easier to add to.
Cyan accent color Keep #22d3ee throughout Same — used for dot indicators, active states, card tags Cyan is part of the brand identity.
Marquee footer Keep Scrolling text strip, font-mono, white/25 Same — above a single ruled line with copy/tagline split The marquee is a good signature element.
Implementation scope
Changes are scoped to portfolio-onepager.tsx and the portfolio-specific CSS block in globals.css. The parallax component, lightbox, and Autopiac are untouched.
FileChange
landing-parallax.tsx No changes
portfolio-onepager.tsx Nav layout, hero layout restructure, archive grid class changes, signals to filmstrip
globals.css (portfolio block) Replace hud-card blur with flat dark surface; new .p-archive-grid, .p-nav, hero layout classes
